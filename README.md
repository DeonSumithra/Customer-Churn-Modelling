# 📉 Customer Churn Modelling

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a customer churn dataset to identify the key factors influencing customer attrition. Using Python and data visualization techniques, the analysis uncovers trends, relationships, and patterns that help understand why customers leave and which attributes have the greatest impact on churn.

## 🎯 Objectives

* Analyze customer demographics and banking attributes
* Identify key factors contributing to customer churn
* Discover trends using statistical and visual analysis
* Generate actionable business insights for customer retention

## 📊 Dataset Overview

The dataset contains customer information such as:

* Credit Score
* Geography
* Gender
* Age
* Tenure
* Balance
* Number of Products
* Credit Card Status
* Active Membership
* Estimated Salary
* Churn Status (Exited)

## 📈 Analysis Performed

* Data Cleaning and Preprocessing
* Univariate Analysis
* Bivariate Analysis
* Correlation Analysis
* Customer Churn Distribution
* Age Group-wise Churn Analysis
* Feature-wise Visualizations
* Key Business Insights Extraction

## 🔍 Key Insights

* Approximately **20.37%** of customers have churned.
* Customers from **Germany** exhibit the highest churn rate.
* **Age** is one of the strongest indicators of customer churn.
* Customers with higher account balances tend to churn more frequently.
* Active members are significantly less likely to leave the bank.
* Product ownership and geographic location have a noticeable impact on retention.
